- 기반기술은 뭐냐 ?

 사용자 프런트: javascript html5, angularjs
 백엔드 : centos,스프링4, mysql, mybatis,java


-관심정보 수집/분석/매칭 

1. 광고주/매체사 광고코드를 해당 사이트에 htm코드 형태로 삽입합니다.
2. 일반사용자가 사이트 접근시에 데이타(유입경로,검색어)가 수집됩니다.
3. 수집된 데이트를 가지고 분석을 통해 사용자의 대표키워드를 만들 만듭니다.
	유입경로 키워드, 상품정보에서 형태소 분석을 통한 키워드 생성==> 여기에서 나온 키워드를 빈도수와 가중치를 적용하여 총합점수가 높은 순으로 
	사용자 대표키워드가 결정됩니다.
	
4. 사용자가 브랜드광고가 삽입된 사이트에 방문시 대표키워드를 가지고 광고 시스템에 등록된 광고키워드를 찾아 타케딩 브랜드광고를 보여줍니다.


-모바일광고 플랫폼 개발내용?

안드로이드앱과 ,ios 앱내 광고영역에 브랜드박스 광고를 삽입하기위해서는  각각 플렛폼에  쉽게 적용할수 
있는 라이브러리 형태로 제작해야합니다. 
 따라서  광고 브랜드박스광고  sdk 를 api 형태로 만들어야 개발자가 자기 앱(안드로이드,ios ) 내에 쉽게 삽입할수 있습니다.
 
 - 잘하는게 뭐냐? 
 개발자로서 
 1. 웹데이타 크럴링
